\# CSV to Database ETP Pipeline



\## Problem
Load raw .csv files into a relational database in a repeatable, reliable way.



\## Setup

* Install Python (3.10++ recommended)
* Install dependencies:

&nbsp;  pip install -r requirements.txt



\## Run

(Coming soon) This project will run an Extract -> Validate -> Transform -> Load pipeline



\## Roadmap

* \[] Connect to database
* \[] Extract .csv
* \[] Validate (schema, nulls, duplicates)
* \[] Transform (types, dates, standardization)
* \[] Load into database
* \[] Logging + run report
